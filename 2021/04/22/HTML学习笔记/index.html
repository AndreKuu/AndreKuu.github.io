盒子模型

盒子模型是css布局的即使，规定了网页元素如何显示，以及元素之间相互关系，css定义所有的元素都可以拥有盒子一样的外形和平面空间，即都包含边框、边界、空白内容区。

margin 边界

在元素外边的空白区域 被称为边界 通常用来控制盒子与盒子之间的距离

margin可以是负值 上边界是margin-top 下边界是margin-bottom 左边界 右边界 但是负值只有top和left方向的值是有效的 因为元素的排列默认是靠左上的

margin 是在元素可视区外的 所以不会影响元素的可是宽高

margin 是在元素可视区外的，所以不会影响背景图片，背景是从元素左上角开始的

margin: 0 auto；是水平居中 左右两边设置auto可以让一个元素水平居中

margin一个值 代表四个方向 两个值 代表上下 左右 三个值代表上 左右 下，四个值代表上右下左

如果给最右边的盒子加了margin-right就会掉下去 这时候标记这个盒子 添加no-margin 就是单独设置 = 0

padding内填充

padding一个值 代表四个方向 两个值 代表上下 左右 三个值代表上 左右 下，四个值代表上右下左

在设定页面中一个元素内容到元素的边缘之间的距离 也称为补白

  • 位于border到content之间

用法：

  • 用来调整内容在容器的位置

  • 用来调整子元素在父元素中的位置 注意 padding的属性需要添加在父元素上

  • paddng的值是额外加载元素原有大小之上的，如果要保证元素大小不变，需要元素宽或高设置的时候减去padding的值

  • 依旧默认在盒子的左上角，而不是content区域，padding不影响背景图片的默认位置

总结：margin是调理盒子盒子之间的距离的，padding是调整内容到容器之间的

置换元素 为其他内容占位置的元素，浏览器根据气元素的标签和属性来决定元素的鸡腿展示内容 典型就是img标签（具有行内块元素的特性）

行内块标签

根据一行可以显示多个，可以进行狂傲设置的特性，我们跳出来如下标签，

行内块标签：

  1. 一行内可以显示多个
  2. 支持宽高的设置
  3. 代码换行会解析空格
  4. 需要设置对齐方式为顶部 vertical-align:top

标签转换

display属性

  • block 块标签
  • inline行内标签
  • inline-block行内块标签
  • list-item将元素转为列表 li的默认类型
  • none 无 （该元素会隐藏起来，经常可以做成下拉菜单）

我们改变的元素的显示方式，而不是元素的本性。

position 定位

属性书写：

postition：static/relative/absolute/fixed

属性详解：

  1. position：static;默认；静态
  2. position：relative相对定位
    特性：

可以利用top、bottom、left、right设置偏移值

位移方向参照自己当前位置

不设置偏移方向，元素不发生位置变化

提升层级 使用了position relative就会提升图层的的等级 覆盖在别的图层上面 

   .item01{
        background: #722;
        position: relative;
        left: 50px;
        top: 70px;
    }

同层Z-index
统计元素情况：

  1. 定位会提高层级 除了static
  2. 后面的定位元素层级，高于前面的定位元素层级

嵌套元素情况：

自己的层级高于父级的层级

该表层级属性：

z-index:1 默认为 0

特性 只能嫁给定位元素 数值越大涂层等级越高 ，可以接受负数图层 建议同级元素之间比较图层 

    .item01{
        background: #722;
        position: relative;
        left: 50px;
        top: 70px;
    }
    .item02{
        background: grey;
        position: relative;
        top: 20px;
        z-index: -1;
    }

absolute 使用了会脱离原来的文档流

特性：
元素脱离文档流

内联标签可以支持宽高

块标签不设置宽高的情况下， 宽高内容撑开

可以用top、bottom、left、right等设置偏移量

位移方向参照伏击位置

一般我们用相对定位作为绝对定位的父级，子绝父相

提升层级

fixed 固定定位

固定在页面可是去的某个位置 类似于background-attachment：fixed

特性：

  • 针对块元素，不设置宽高的时候 宽高由内容撑开
  • 是内联、行内元素支持宽高
  • 元素定位参照：页面可视区（即使滚动条滚动，也不影响到可视区哦）
  • 提升层级
  • 脱离文档流

圆角

border-radius数值 有三种表示方法：px、%、em

border_radius 可以设置水平和垂直方向的半径，若省略垂直方向的值，则表示垂直和水平方向的半径值一样，半径x/y

border-radius: 10px;

border-radius有几种参数输入方式 

        border-radius: 10px/40px;
        border-radius: 10px 30px;
        border-radius: 10px 20px 30px;

一个值则四个角都一样，两个值是（左上角&右下角 右上角&左下角）
三个值（左上角 右上角&左下角 右下角） 四个值 （左上角 右上角 右下角 左下角）

            border-radius: 10px/40px;
        border-radius: 10px 30px;
        border-radius: 10px 20px 30px;
        border-radius: 10px 20px 30px 40px/20px 40px 50px 60px;
        border-top-left-radius: 30px;
        border-bottom-right-radius: 60px;

伪元素

利用css给一个标签创建出来的自己元素，该元素在页面中真实存在，但是在html中并不存在

两个关键字 before after

before往指定的标签内部的前端添加内容

after 往指定的标签的内部的后端添加内容

before after都需要利用content添加内容，所有的样式针对该内容进行设置

content可以添加的内容 文字、调用当前元素的属性、url()引用没提文件、添加图标 

<style>
        div{
            color: red;

        }
        div::before{
            content: "hello";
            color: green;
        }
        div::after{
            content: " and C++";
            color: blue;
        }
    </style>
</head>
<body>
    <div> python</div>
</body>

显示的内容一部分是从div里的content实现的，不在html内，这个就是伪元素

还可以引用当前元素的属性 这里调用了a类的地址content显示 

<style>
        a::after{
            content: attr(href);
        }
    </style>
</head>
<body>
    <a href="http://www.baidu.com/"></a>
</body>

还可以引用url()

<style>
        div::after{
            content:url(../../images/1.gif)
        }
    </style>
</head>
<body>
    <div>芜湖</div>
</body>
